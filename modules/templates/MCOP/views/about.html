About

Sahana is a component of the Washington Common Operating Picture (WA-COP) platform for use in both large scale disaster management and coordination of day-to-day activities and operations.

Sahana integrates and shares data with other systems to provide multistakeholder situational awareness:

  • Tasks from Wrike
  • Tasks are mapped and published to CommandBridge

{{=T("Incidents")}}

Sahana can be used to manage Incidents

{{=T("Alerts")}}

Sahana can be used to manage Alerts

{{=T("Tasks")}}

Sahana can be used to manage Tasks. These tasks can be imported from Wrike and mapped within Sahana

{{=T("Map")}}

Sahana's map provides situational awareness of all information

{{=T("Resource Inventory")}}

Sahana can be used to manage Resource Inventories

{{=T("Responding Resources")}}

Sahana can be used to manage Resources Responding to Incidents

{{=T("Contact Directory")}}

Sahana can be used to manage a Contact Directory

{{=T("Stakeholders")}}

Sahana can be used to manage Stakeholders

{{=T("Facilities")}}

Sahana can be used to manage Facilities
